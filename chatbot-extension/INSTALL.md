# Инструкция по установке AI Customer Support Chatbot

## Шаг 1: Подготовка иконок

Перед установкой создайте иконки для расширения:

1. Перейдите в папку `icons/`
2. Прочитайте `README.txt` для инструкций
3. Создайте файлы: `icon16.png`, `icon48.png`, `icon128.png`

**Быстрый вариант:** скачайте любые PNG иконки нужных размеров.

---

## Шаг 2: Загрузка в Chrome (для тестирования)

1. Откройте Chrome
2. Перейдите на `chrome://extensions/`
3. Включите **"Режим разработчика"** (переключатель в правом верхнем углу)
4. Нажмите **"Загрузить распакованное расширение"**
5. Выберите папку `chatbot-extension`
6. Расширение появится в списке

---

## Шаг 3: Настройка

1. Нажмите на иконку расширения в панели Chrome
2. Введите ваш OpenAI API ключ (получить на https://platform.openai.com/api-keys)
3. Нажмите "Сохранить ключ"

---

## Шаг 4: Обучение на сайте

1. Перейдите на сайт, который хотите обучить
2. Нажмите на иконку расширения
3. Нажмите "Обучить на этом сайте"
4. Дождитесь завершения парсинга

---

## Шаг 5: Использование чата

После обучения на сайте появится виджет чата в правом нижнем углу.
Нажмите на него и задавайте вопросы!

---

## Публикация в Chrome Web Store

### Подготовка

1. Создайте аккаунт разработчика: https://chrome.google.com/webstore/devconsole/
2. Оплатите разовый взнос $5
3. Подготовьте:
   - Описание расширения (на английском)
   - Скриншоты (1280x800 или 640x400)
   - Промо-изображения (опционально)
   - Политику конфиденциальности

### Загрузка

1. Создайте ZIP архив папки `chatbot-extension`
2. В Chrome Web Store Developer Dashboard нажмите "Новый элемент"
3. Загрузите ZIP файл
4. Заполните все поля
5. Отправьте на проверку

### Проверка

- Обычно занимает 1-3 рабочих дня
- Могут запросить дополнительную информацию
- После одобрения расширение появится в магазине

---

## Настройка Stripe (для монетизации)

1. Создайте аккаунт на https://stripe.com
2. Перейдите в Products → Add product
3. Создайте продукт "AI Chatbot Pro" с ценой $9.99/месяц (recurring)
4. Перейдите в Payment Links → Create
5. Скопируйте ссылку
6. Замените `STRIPE_PAYMENT_LINK` в файле `popup/popup.js`

---

## Решение проблем

### Расширение не загружается
- Проверьте, что все файлы на месте
- Убедитесь, что иконки существуют
- Посмотрите ошибки в `chrome://extensions/`

### Виджет не появляется
- Убедитесь, что API ключ сохранён
- Обучите сайт перед использованием
- Проверьте консоль браузера (F12) на ошибки

### Ошибки API
- Проверьте баланс на аккаунте OpenAI
- Убедитесь, что ключ правильный (начинается с `sk-`)
- Проверьте подключение к интернету

## Структура файлов

```
chatbot-extension/
├── manifest.json           # Конфигурация расширения
├── popup/
│   ├── popup.html         # UI настроек
│   ├── popup.js           # Логика настроек
│   └── popup.css          # Стили
├── content/
│   ├── content.js         # Виджет чата
│   ├── parser.js          # Парсинг страниц
│   └── widget.css         # Стили виджета
├── background/
│   └── service-worker.js  # Фоновая логика, API
├── utils/
│   ├── storage.js         # Работа с хранилищем
│   └── openai.js          # OpenAI API
└── icons/
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```
